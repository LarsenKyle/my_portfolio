About Me

BIO

I am a Front-End web developer based in Grand Rapids Michigan. I have a passion for designing interesting, intuitive, and interactive user interfaces. Innovation and creativity come together to create user focused designs that will manage the specific needs of whoever I’m working with

User Interface//

A good user interface provides a "user-friendly" experience, allowing the user to interact with the software or hardware in a natural and intuitive way.

User Experience//

I consider a person's emotions and attitudes about using a particular product, system or service. It includes the practical, experiential, affective, meaningful and valuable aspects of human–computer interaction and product ownership.

Responsive Design//

The goal of responsive design is to build web pages that detect the visitor's screen size and orientation and change the layout accordingly.
